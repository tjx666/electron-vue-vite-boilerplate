# visual-craft-station

## 开发

### 安装依赖

项目使用 monorepos 的组织方式，包管理器使用 pnpm，pnpm 内置对 monorepos 的支持。

```bash
pnpm install
```

### 启动客户端

```bash
pnpm app
```

### 调式

工具箱项目添加了一些开发时使用的快捷键：

- Toggle DevTools: <kbd>Cmd</kbd> <kbd>Alt</kbd> <kbd>I</kbd> or <kbd>F12</kbd>
- Reload: <kbd>Cmd</kbd> <kbd>R</kbd> or <kbd>F5</kbd>
